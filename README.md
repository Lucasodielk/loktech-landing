# LOK TECH — Site de apresentacao

Landing page da LOK TECH (marketing / criacao de sites). Pagina estatica, 100% autossuficiente (um unico index.html com CSS e imagens embutidas).

## Como publicar na Vercel
1. Acesse https://vercel.com e faca login com o GitHub.
2. "Add New... > Project" e selecione este repositorio.
3. Framework Preset: **Other**. Nao precisa build. Clique em **Deploy**.
4. Em segundos a Vercel gera a URL publica do site.
